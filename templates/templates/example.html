模板示例和使用说明

如何创建和使用草稿模板

草稿模板可以帮助你快速生成具有特定风格的剪映草稿，以下是创建和使用模板的步骤：

1. 模板文件结构

模板文件是一个JSON格式的文件，包含以下主要部分：

  • name：模板名称
  • template：模板基本配置
  • font_style：字体样式设置
  • keyframes：关键帧动画设置
2. 完整的模板示例
{
  "name": "渐入缩放模板",
  "template": {
    "effect_type": "渐入",
    "font_type": "抖音美好体"
  },
  "font_style": {
    "id": "7244518590332801592",
    "path": "抖音美好体.ttf",
    "size": 10,
    "color": [1, 1, 1]
  },
  "keyframes": [
    {
      "property_type": "KFTypePositionX",
      "keyframe_list": [
        {
          "curveType": "Line",
          "time_offset": 0,
          "values": [-0.8]
        },
        {
          "curveType": "Line",
          "time_offset": 3000000,
          "values": [0]
        }
      ]
    },
    {
      "property_type": "KFTypeScale",
      "keyframe_list": [
        {
          "curveType": "Line",
          "time_offset": 0,
          "values": [0.8]
        },
        {
          "curveType": "Line",
          "time_offset": 3000000,
          "values": [1.0]
        }
      ]
    },
    {
      "property_type": "KFTypeOpacity",
      "keyframe_list": [
        {
          "curveType": "Line",
          "time_offset": 0,
          "values": [0.2]
        },
        {
          "curveType": "Line",
          "time_offset": 1000000,
          "values": [1.0]
        }
      ]
    }
  ]
}
3. 字体样式说明

字体样式定义了文本素材的外观：

  • id：字体ID，通常可以使用默认值
  • path：字体文件路径，指定要使用的字体
  • size：字体大小，数值越大文字越大
  • color：字体颜色，使用RGB值的数组，范围是0到1

字体颜色示例：

  • 白色：[1, 1, 1]
  • 黑色：[0, 0, 0]
  • 红色：[1, 0, 0]
  • 绿色：[0, 1, 0]
  • 蓝色：[0, 0, 1]
4. 关键帧动画说明

关键帧定义了素材随时间变化的动画效果：

  • property_type：属性类型，指定要调整的属性
  • keyframe_list：关键帧列表，定义不同时间点的属性值

常用的属性类型有：

  • KFTypePositionX：X轴位置，范围通常为-1到1
  • KFTypePositionY：Y轴位置，范围通常为-1到1
  • KFTypeScale：缩放，1表示原始大小
  • KFTypeRotation：旋转，以角度为单位
  • KFTypeOpacity：透明度，0表示完全透明，1表示完全不透明

每个关键帧包含：

  • curveType：曲线类型，通常为"Line"线性变化
  • time_offset：时间偏移，以微秒为单位
  • values：属性值，根据属性类型有不同的含义
5. 如何添加到系统

创建好模板文件后，有两种方式可以添加到系统：

  1. 通过管理员界面上传：在模板管理页面点击"上传模板"按钮
  2. 手动添加：将模板文件放到 temp/templates/ 目录下

系统会自动检测和加载所有位于 temp/templates/ 目录下的JSON文件作为模板。

6. 常见问题
  • Q: 为什么我的模板没有被加载？
    A: 请确保JSON格式正确，并且包含所需的字段。可以使用在线JSON校验工具检查格式。
  • Q: 时间单位是什么？
    A: 时间单位是微秒，1秒 = 1,000,000微秒。例如3秒就是3000000。
  • Q: 如何创建淡入淡出效果？
    A: 可以使用KFTypeOpacity属性，设置从0到1的变化来创建淡入效果。
